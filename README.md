## Installation

To install and run this project, follow these steps:

1. **Clone the repository:**

   ```bash
   git clone https://github.com/helinlazz/Projects.git
   ```

   2. **Navigate to the project directory:**

   ```bash
   cd projects
   ```

2. **Install dependencies:**

   ```bash
   npm install
   ```

## Usage

Start the application:

```bash
npm run dev
```
